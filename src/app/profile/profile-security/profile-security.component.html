Security

More advanced security options to increase your account's security

Enable 2FA Email

When you sign in on a new device or from a new location, we'll send you a security code to your email to enter on the sign-in page.

Enable Application 2FA

When you sign in on a new device or from a new location, we'll send you a security code to your chosen 2FA mobile application (such as Google Authenticator) to enter on the sign-in page.
